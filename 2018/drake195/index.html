Course Description

Mooreâ€™s law (i.e. frequency scaling) for processors ended in early 2000s; since then, computer architectures have become massively parallel. Unfortunately, developing systems which take advantage of these parallel machines is significantly harder than writing single-core applications. This course is an introduction to multithreaded and distributed systems programming and will equip students with the basic tools necessary to write high performance parallel systems. Using C++11 std::thread and Boost.Asio as our foundation, we will explore modern software patterns and parallel programming paradigms which aid in producing correct, high quality, reusable parallel and distributed systems.
